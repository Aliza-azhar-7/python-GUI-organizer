# Python GUI Organizer

A Python-based file organizer with a graphical user interface (GUI) that automatically sorts files into separate folders such as Videos, Images, and Documents with a single click.

## Features
- Easy-to-use GUI
- One-click file organization
- Automatically creates folders
- Sorts files based on their type

## Technologies Used
- Python
- Tkinter

## How to Run
1. Download the project files.
2. Run `folder_organizer_gui.py`.
3. Select the folder you want to organize.
4. Click the organize button.
